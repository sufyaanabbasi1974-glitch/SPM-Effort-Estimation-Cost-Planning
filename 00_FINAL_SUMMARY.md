## 🆘 IF YOU GET STUCK

### Local Issues
```
Problem: Server won't start
Solution: Check Python version, try: pip install -r requirements.txt

Problem: Port 5000 in use
Solution: Change port in app.py or: netstat -ano | find "5000"

Problem: Database error
Solution: Delete database/spm_estimation.db and restart
```

### Live Issues
```
See: LIVE_DEPLOYMENT_QUICK_START.md → Troubleshooting section
Or: COMPLETE_DEPLOYMENT_GUIDE.md → Troubleshooting section
```
